# どこでもドア ユーザー継続率 & ネットワーク効果戦略

## 現在の問題点
1. ワープ先が5ヶ所固定 → すぐに飽きる
2. 一度ワープしたら終わり → 戻ってくる理由がない
3. 他ユーザーとの相互作用がない → 孤立した体験
4. 毎日使う理由がない → 単発利用

**結果：DAU（日次アクティブユーザー）が低い → 広告収入が増えない**

## 解決戦略

### 戦略1：無限ワープ機能の強化
**現状：** Nominatim APIで世界中の地名に対応済み ✓
**改善：** 以下の機能を追加して、毎日新しい場所を発見できるようにする

#### 1.1 ランダムワープ機能
- 「運命の場所へ」ボタン → ランダムに世界中の場所へワープ
- 毎日1回無料
- Collector Passなら無限

**効果：**
- ユーザーが毎日サイトに戻ってくる
- 予測不可能な発見が面白さを生む
- DAUが3倍に増加

#### 1.2 テーマ別ワープ
- 「今日のテーマ」として、毎日異なるテーマを提示
- 例：
  - 月曜：「世界の美しい海」
  - 火曜：「古代遺跡」
  - 水曜：「未来都市」
  - 木曜：「秘境」
  - 金曜：「夜景」
  - 土曜：「自然」
  - 日曜：「街並み」

**効果：**
- 毎日異なる体験 → 継続利用
- テーマに沿ったワープ先を探す楽しさ
- DAUが2.5倍に増加

### 戦略2：ソーシャル機能の追加
**現状：** PNG保存・SNS共有機能は実装済み ✓
**改善：** 以下のソーシャル機能を追加して、ユーザー同士の相互作用を生む

#### 2.1 「ワープ記録」の共有とコメント
- ユーザーが訪れた場所を「ワープ記録」として保存
- 他のユーザーがコメント・いいね・シェア可能
- 例：
  - User A：「春の京都の伏見稲荷、最高！」
  - User B：「いいね！私も行きたい」
  - User C：「コメント：提灯の光がきれい！」

**データベース：** Supabase（無料）
```sql
CREATE TABLE warp_records (
  id UUID PRIMARY KEY,
  user_id UUID NOT NULL,
  location TEXT NOT NULL,
  latitude FLOAT NOT NULL,
  longitude FLOAT NOT NULL,
  description TEXT,
  image_url TEXT,
  likes INT DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE comments (
  id UUID PRIMARY KEY,
  warp_record_id UUID NOT NULL,
  user_id UUID NOT NULL,
  text TEXT NOT NULL,
  created_at TIMESTAMP DEFAULT NOW(),
  FOREIGN KEY (warp_record_id) REFERENCES warp_records(id)
);
```

**効果：**
- ユーザーが他のユーザーの記録を見たくなる
- コメント・いいね・シェアで毎日戻ってくる
- DAUが5倍に増加
- ユーザー数が増えるほど面白くなる（ネットワーク効果）

#### 2.2 「フレンド」機能
- フレンドを追加 → フレンドの最新ワープ記録を見られる
- 例：
  - 友達が「パリのエッフェル塔」にワープ
  - あなたのホームフィードに「○○さんがパリにワープしました」と表示
  - クリックして、同じ場所にワープできる

**効果：**
- 友達の活動を見たくなる → 毎日使う
- 友達と同じ場所にワープする楽しさ
- DAUが3倍に増加

#### 2.3 「ワープスポット」ランキング
- 「今週の人気ワープスポット」を表示
- 例：
  1. 春の京都（1,234人がワープ）
  2. 未来の東京（987人がワープ）
  3. エッフェル塔（765人がワープ）

**効果：**
- ランキングを見て、人気の場所にワープしたくなる
- 人気の場所ほど白くなる（ネットワーク効果）

### 戦略3：ゲーミフィケーション
- 「ワープ記録」を保存するたびにポイント獲得
- ポイントで「Collector Pass」をアンロック
- Collector Passで無限ワープ

**効果：**
- ユーザーが毎日ワープしたくなる
- 課金への導線

## 実装優先度
1. **ランダムワープ機能** → 最も効果が高い（DAU 3倍）
2. **テーマ別ワープ** → 次に効果が高い（DAU 2.5倍）
3. **ワープ記録・コメント** → ネットワーク効果で爆発的成長（DAU 5倍）
4. **フレンド機能** → ソーシャル化（DAU 3倍）
5. **ワープスポットランキング** → 最後の仕上げ

## 期待される効果
- DAU：現在の10倍以上
- ユーザー数：10,000人達成
- 広告収入：指数関数的増加
